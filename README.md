# php-fpm

This image is based on official php:7.4-fpm-alpine. Because the official image doesn't have popular extensions we add some of them to the image.

The extensions are listed below:
* php-xmlrpc
* php-mbstring
* php-zip 
* php-intl 
* php-soap
* php-gd
* php-opcache
* php-pdo
* php-mysqli
* php-pdo_mysql
* php-pgsql
* php-pdo_pgsql
* php-bcmath
* php-bz2
